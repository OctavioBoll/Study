¿Como Funciona?

Es una herramienta para organizar y ayudar a estudiar diferentes temas, organizando con materias que contienen unidades y cada unidad tiene contenido los temas con una descripcion.

Formas de uso

  • Se puede usar como predeterminado ingresando la Materia, Unidades y Temas.
  • Se puede usar como una secuencia de pasos a seguir, describiendo los pasos en la pestaña "Temas".
  • Otra forma de uso es el tener un item y que cada item tenga una descripcion.
